THE LAB REPTILES - DOCUMENTS FOLDER
====================================

This folder contains downloadable documents for customers and visitors.

FOLDER STRUCTURE:
-----------------

/caresheets/
  - gargoyle-gecko-care-guide.pdf    (Gargoyle Gecko comprehensive care guide)
  - ball-python-care-guide.pdf       (Ball Python care guide - coming soon)

/new-owner/
  - transition-guide.pdf             (Guide for bringing your new reptile home)
  - feeding-schedule.pdf             (Printable feeding schedule template)
  - health-checklist.pdf             (Health monitoring checklist)


HOW TO ADD NEW DOCUMENTS:
-------------------------
1. Create your PDF document
2. Name it descriptively (use-lowercase-with-dashes.pdf)
3. Place it in the appropriate subfolder
4. Update the caresheets.html page to include a link to the new document


FILE NAMING CONVENTION:
-----------------------
- Use lowercase letters
- Use dashes (-) instead of spaces
- Include the document type in the name
- End with .pdf

Examples:
  - gargoyle-gecko-care-guide.pdf
  - ball-python-feeding-chart.pdf
  - new-owner-checklist.pdf
